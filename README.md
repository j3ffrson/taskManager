# 📝 To-Do API

API REST desarrollada con Spring Boot para la gestión de tareas.  
Este proyecto sigue buenas prácticas de desarrollo profesional: uso de issues, ramas por feature, commits estructurados y versionado progresivo.

---

## 🚀 Tecnologías

- Java 25
- Spring Boot
- Spring Web
- Spring Data JPA
- Lombok
- Base de datos: PostgreSQL

---

## 📌 Estado del proyecto

🔧 En desarrollo — versión inicial (v1.0)

---

## 🎯 Funcionalidades (v1.0)

- Crear tareas
- Listar tareas
- Actualizar tareas
- Eliminar tareas
- Filtrar tareas por estado
